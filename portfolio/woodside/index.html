Woodside
72-01 Queens Boulevard, Woodside
project details
site

Woodside is a $180 million ground up mixed use and mixed income rental project with a $100 million hard cost budget. The project is 380,000 SF, 12 stories, 364 residential units of which 109 will be set aside as affordable housing via the voluntary inclusionary housing program in addition to 26,000 SF of ground floor retail.
